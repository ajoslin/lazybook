# lazybook 

> Create async-loaded apis that aren't async from the outside

Experimental Webpack module for a talk. Use at your own risk.

## Install

```
$ npm install --save lazybook
```


## Usage

```js
var lazybook = require('lazybook')

lazybook('input')
//=> output
```
